# Class 선언하기  

## Class  
- Class는 Field와 Method를 갖는다.
- Field: Class의 속성.
- Method: Class의 기능.

## Class 선언
### [접근제한자] class [class이름]
- ex_ public class Hello      
- !!! 클래스의 이름은 항상 대문자 !!!
```java
public class Hello {
    public static void main(String[] args) {
        // 필드
        // 생성자
        // 메소드 
        System.out.println("true = " + true);
    }
}
```
